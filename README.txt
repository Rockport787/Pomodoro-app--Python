this is a tkinter GUI project which keeps track of a healthy work schedule

features

1.it allots 25 min of work timer followed by a 5 min break time 

2.After 4 work cycles(25 work+5 break) which amounts to 30min*4= 2 hours
  the timer gives you a break time of 25 min 

3. values can be customised easily via the global variables provided
